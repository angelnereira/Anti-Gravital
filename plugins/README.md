# Official WASM Plugins

This directory will contain official WASM plugin binaries for Anti-Gravital.

Plugins ship as `.wasm` files and are loaded at startup by `ag-wasm-host`.
The plugin registry will be available at plugins.antigravital.dev in Phase 5.

## Planned Plugins

- `prometheus.wasm` — Prometheus metrics export
- `datadog.wasm` — Datadog APM integration
- `sentry.wasm` — Sentry error tracking
